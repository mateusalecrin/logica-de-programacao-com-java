# Tradução

<p> De início usei um código próprio feito em Portugol e o traduzi para o Java, ou seja, português para o inglês.
  O exercício pede ao usúario para que insira 10 números somente positivos, os recebe dentro da estrutura de repetição For e já os valida pedindo para informar novamente caso sejam números negativos ou neutro (0). O algorítimo armazena estes números e retorna ao usuário mostrando-lhe um menu de opções com as seguintes possibilidades: Todos os valores, somente o primeiro valor, somente o último valor, somente os valores pares, o maior valor. 
</p>
