<h1 align="center"> O primeiro contato</h1>

<p>De início usei um código próprio feito em Portugol e o traduzi para o Java, ou seja, português para o inglês.</p>

<br/>

## Exercício
<ol>
 <li> O exercício pede ao usúario para que insira 10 números somente positivos, os recebe dentro da estrutura de repetição For e já os valida pedindo para informar novamente caso sejam números negativos ou neutro (0). O algorítimo armazena estes números e retorna ao usuário mostrando-lhe um menu de opções com as seguintes possibilidades: 
  <ul>
  <li> Todos os valores </li>
  <li> Somente o primeiro valor </li>
  <li> Somente o último valor </li>
  <li> Somente os valores pares </li>
  <li> O maior valor. </li> 
  </ul>
</li>
</ol>
