<h1 align="center"> Lista 03 </h1>

<p>Nesta parte da trilha começamos a dividir as tarefas em métodos diferentes, sendo responsabilidade do aluno identificar se o método precisa receber parâmetros e retorno.</p>

<br/>

## Exercícios

<ol>
<li> Solicita o ano de nascimento de um usuário e informa sua idade no ano de 2021. Não sendo necessário considerar se o usuário já fez ou não aniversário.</li>

<li> Recebe três notas de um aluno e solicita ao professor qual média ele deseja calcular: Aritmética ou Ponderada? 
Se for a média ponderada, os pesos de cada nota devem ser informados pelo professor. Cada média é calculada por um método e os dois métodos retornam a média. Outro método é criado para receber a média final e mostrar se o aluno está aprovado ou reprovado, considerando que a média mínima para aprovação é 7.</li>

<li> Programa que a partir do valor de uma compra, faz o cálculo de um desconto para o usuário. Até R$100 sem desconto, o valor permanece o mesmo. De R$100,01 a R$200 - desconto de 20%. Acima de R$200 - desconto de 30%.</li>
</ol>

<br/>

## Conceitos abordados nesta trilha: 
<ul>
<li> Estruturas de armazenagem (Vetor e matriz); </li>
<li> Estruturas condicionais (If / else, switch / case); </li>
<li> Estruturas de repitição (For, while, do while); </li>
<li> O método parseInt(); </li>
<li> O método equals(); </li>
<li> A classe JOptionPane; </li>
<li> A classe DecimalFormat; </li>
</ul>
