<h1 align="center"> Lista 02 </h1>

## Exercícios

<ol>
<li> Um programa que recebe um vetor de valores e mostra a quantidade de valores negativos. Os valores e o tamanho do vetor é informado pelo usuário. </li>
<li> Um programa que recebe quatro notas do aluno e as insere em um vetor. Depois, calcula a média aritmética entre as quatro notas e mostra o "conceito" do aluno conforme as instruções: 
    <ul>
    <li> 9.0 ou maior = A </li>
    <li> Entre 8.0 e 8.9 = B </li>
    <li> Entre 7.0 e 7.9 = C </li>
    <li> Menor que 7.0 = D </li>
    </ul>
</li>
<li> Recebe um número inteiro e apresenta um menu ao usuário:
    <ul>
    <li> Verificar se o número é múltiplo de algum outro número (Solicitado ao usuário) </li>
    <li> Verificar se o número é par </li>
    <li> Verificar se o número está entre 0 e 1000 </li>
    <li> Sair </li>
    </ul>
</li> 
Após a opção escolhida é informado os dados e retorna ao menu.
<li> Permite ao usuário informar uma quantidade de latão em quilos e fornece o total de cobre e zinco necessários para fabricar essa quantidade. (Sabe-se que o latão é constituído de 70% de cobre e 30% de zinco). </li>
<li> Solicita dados aos usuários que frequentam um determinado clube. São eles:
    <ul>
    <li> Idade </li>
    <li> Fumante (S ou N) </li>
    <li> Salário </li>
    <li> A quanto tempo frequenta o clube (meses) </li>
    </ul>
    A seguir informa a média das idades e a média dos salários dos frequentadores. Também quantos são fumantes e quantos não são, e a porcentagem de pessoas que frequentam o clube há mais de 03 meses.
</li>
<li> Armazena em um vetor o valor total das vendas de cada um dos 12 meses do ano de uma empresa. E a seguir disponibiliza um menu que forneçe os seguintes dados:
    <ul>
    <li> O mês com a maior venda </li>
    <li> O mês com a menor venda </li>
    <li> A média das vendas do ano todo </li>
    <li> A média das vendas dos meses pares </li>
    <li> A média das vendas do segunto semestre </li>
    </ul>
</li>
<li> Recebe em uma matriz notas em 03 disciplinas de um aluno nos quatro bimestres, podendo aceitar até duas casas decimais. A seguir disponibiliza um menu com as seguintes opções:
    <ul>
    <li> Todas as notas de todas as disciplinas </li>
    <li> Qual a maior nota, e em qual disciplina foi </li>
    <li> A média das notas de uma discplina (é solicitado qual) </li>
    <li> As notas de um dos bimestres (é solicitado qual) </li>
    </ul>
</li>
</ol>

<br/>

## Conceitos abordados nesta trilha: 
<ul>
<li> Estruturas de armazenagem (Vetor e matriz); </li>
<li> Estruturas condicionais (If / else, switch / case); </li>
<li> Estruturas de repitição (For, while, do while); </li>
<li> O método parseInt(); </li>
<li> O método equals(); </li>
<li> A classe JOptionPane; </li>
<li> A classe DecimalFormat; </li>
</ul>
